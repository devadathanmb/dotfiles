## Quick Setup

- Install the needed tools
```bash
paru -S i3-gaps \
polybar\
rofi\
nitrogen\
picom\
betterlockscreen\
zscroll\
nm-applet\
playerctl\
maim\
xrandr\
dunst\
xclip
```

- Setup the symlinks  
```bash
git clone https://github.com/devadathanmb/dotfiles.git
cd dotfiles
./install
```
